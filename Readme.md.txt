Descripción de Proyecto Ready2Wear:
Ready2Wear es una app que permitirá a los usuarios alquilar trajes entre los usuarios, de forma sencilla y eficiente. La plataforma conectará a propietarios y arrendatarios, ofreciendo una interfaz amigable, la cual muestre los trajes con fotos, descripción, precio, talla y disponibilidad, etc. Tendrá una búsqueda avanzada con filtros y geolocalización mediante la API de Google Maps, reservas de reservas de artículos(con una fianza obligatoria), pagos a través de PayPal, y comunicación directa entre usuarios mediante chat.
El proyecto lo desarrollaré completamente en Firebase, que se encargará de ofrecer features básicas y generales como: la autenticación de usuarios, almacenamiento de datos, notificaciones, gestión de la base de datos en tiempo real y por lo tanto  permitiendome un desarrollo rápido y escalable. Además, Ready2Wear permitirá hacer valoraciones y puntuaciones, favoritos, suscripciones para destacar publicaciones y otras funcionalidades exclusivas.
En conclusión, Ready2 ofrecerá una solución completa y confiable para el alquiler de trajes, ya que debido a los altos precios de estos artículos, se ha convertido inaccesible para la mayoría de la población. Por ello optamos por empezar a utilizar esta app que ayudará a muchas personas generar ingresos, a otros ahorra y por último a tener un menor impacto ambiental
